It's a "has-a" relationship. # -----------------------------------------example:------------------------------- # A TM (Technical Mentor) has students # But the students also exist on their own — they are not part of TM directly # Code Breakdown # -------------------------1. ------------------------------3. add_students method----------------------------------------------- # def add_students(self, newStudents): # self.students.extend(newStudents) # This method takes a list of new students and adds them to the mentor's students list using extend(). #
